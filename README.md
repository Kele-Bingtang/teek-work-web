# Teek Work Web

`Teek Work Web` 是一个企业级的 PC 端项目，有 UAC（User Account Control）、AG（API Generator）、FS（File Store）等前端子项目。

该项目需要搭配后端才能运行，后端项目仓库：[Teek Work Platform](https://github.com/Kele-Bingtang/teek-work-platform)。

## 本地开发

安装依赖

```sh
pnpm install
```

启动 UAC 项目

```sh
pnpm uac dev
```

启动 AG 项目

```sh
pnpm ag dev
```

启动 FS 项目

```sh
pnpm file dev
```
